Class Filter

Interface containing a Signature and ability to "match" it with others.

This class is intended to be handled exclusively by the Context and not by any users. However, using it outside "that context" can be trivial. Filters require a list of *known* or *valid* components to check against, and this list can be obtained through the Context's `getComponentList()` method.
